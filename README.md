# 🧠 TradeSense – AI-Powered Forex Signal Dashboard

TradeSense is an intelligent trading assistant for the MetaTrader 5 platform. It uses technical indicators, machine learning, and custom rule-based strategies to generate real-time trade signals through a beautiful GUI powered by PyQt5.

---

## ✨ Features

- 📡 **Live MT5 Signal Analysis**
- 🧠 **AI/ML Trade Signal Prediction**
- 📈 **Equity + Drawdown Curve Backtesting**
- 🔔 **Alerts via Email & Telegram**
- 🗂 **Auto Trade Logging (CSV)**
- ⚙️ **Strategy Builder (Rule-based Logic)**
- 🧪 **Backtest Mode with SL/TP Simulation**
- 🌙 **Dark Mode + HUD Toggle**
- 🌐 **Multi-language Scaffold (EN/HI)**
- 🛠 **Customizable Parameters (Threshold, Cooldown, SL/TP)**
